#Installation

All commands must be executed in Prompt Comand at the project folder.

##Install database tables
```
php artisan migrate:install
```

##Install data example
```
php artisan db:seed
```
#Execute

All commands must be executed in Prompt Comand at the project folder.

##Start server
```
php artisan serve
```
